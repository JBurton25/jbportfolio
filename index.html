Pokemon Memory Game

This is a Memory game created with React. The application's UI is split into five components. The app keeps track of the player's score (one point for each unique Pokemon selected). Every time a Pokemon is clicked, the page reshuffless in a random order. If the player clicks on all 12 unique images, without clicking on the same one twice, they receive an alert they win. Otherwise, the game restarts after an incorrect selection.

Friend Finder is a compatibility-based application -- basically a dating app. This full-stack site will take in results from your users' surveys, then compare their answers with those from other users.
